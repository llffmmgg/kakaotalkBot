# PRD: 개발자/IT 채용공고 카카오톡 오픈채팅 봇

## 목표
개발자/IT 구직자가 모인 카카오톡 오픈채팅방에, 최신 채용공고를 자동으로 알려주고
원할 때 검색할 수 있게 한다 — 사용자가 여러 채용 사이트를 직접 돌아다니지 않아도 되게.

## 사용자
- 이직/구직 중인 개발자(백엔드·프론트·모바일·데이터 등)가 모인 오픈채팅방 참여자.
- 방장(봇 운영자): 전용 안드로이드 폰에서 메신저봇R을 돌린다.

## 핵심 기능
1. **자동 푸시**: 새 채용공고가 올라오면 채팅방에 자동으로 전송(중복 없이).
2. **명령어 검색**: `!채용 백엔드 지역:서울 경력:신입` 형태로 직무/지역/경력/고용형태 필터 검색.
3. **필터**: 직무·키워드 / 지역 / 경력·연차 / 고용형태 4종.

## MVP 제외 사항
- 원티드·점핏 등 크롤링(공개 API 없음 → Selenium 필요 → 무료 서버리스 부적합). 후속 검토.
- 카카오 공식 채널/플러스친구 연동(오픈채팅 자동푸시 미지원).
- 개인 맞춤 알림(사용자별 구독 키워드), 지원하기/북마크 등 상호작용.
- 웹 UI/관리자 대시보드(데이터는 `data/jobs.json` 정적 파일로 충분).

## 데이터 소스
- **사람인 공식 JSON API** (`oapi.saramin.co.kr/job-search`). requests-only → GitHub Actions에서 안정적으로 호출. 1일 500회 제한.
- 수집 주기: GitHub Actions cron 6시간마다.

## 비고 (운영 제약)
- 메신저봇R은 비공식 수단 — 카카오톡 업데이트 시 동작이 깨질 수 있고, 폰을 24시간 켜둬야 한다.
- 오픈채팅 자동 전송은 현실적으로 메신저봇R이 유일한 경로.
