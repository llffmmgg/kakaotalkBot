# 카톡 메시지 포맷 가이드

이 프로젝트엔 웹/앱 UI가 없다. "UI"는 카카오톡 채팅방에 출력되는 **텍스트 메시지**다.
모든 봇 출력(`bot/modules/format.js`)은 이 가이드를 따른다.

## 원칙
1. **스캔 가능해야 한다.** 채팅방은 빠르게 흐른다. 한눈에 직무·회사·지역·마감이 들어와야 한다.
2. **짧게.** 카톡 메시지는 길면 잘리거나 "더보기"로 접힌다. 공고 1건은 5~6줄 이내.
3. **링크는 마지막 줄에.** 클릭 가능한 URL은 메시지 맨 아래 한 줄로.
4. **장식 최소.** 과한 이모지·구분선 남발 금지. 정보 우선.

## 하지 마라
| 금지 | 이유 |
|------|------|
| 한 메시지에 공고 10건씩 욱여넣기 | 잘림 + 스팸처럼 보임. 검색 결과는 상위 N건(기본 5)만 |
| 이모지 도배(줄마다 3개+) | 가독성 저하, 봇이 시끄러워 보임 |
| 긴 회사 소개/공고 본문 복붙 | 채팅방 도배. 요약 + 링크로 |
| ASCII 박스/표 그리기 | 카톡 폰트는 고정폭 아님 → 깨짐 |
| 마감 지난 공고 노출 | collector가 거르지만 포맷 단계에서도 방어 |

## 공고 1건 포맷 (권장)
```
📌 {title} — {company}
📍 {region}  |  {career}  |  {employment}
⏰ ~{deadline}
🔗 {url}
```
- 값이 없는 줄(예: deadline 없음)은 통째로 생략한다. 빈 라벨을 남기지 마라.
- `title`이 너무 길면 적당히 잘라 `…` 처리.

## 검색 결과 (여러 건)
- 헤더 1줄 + 공고 블록들. 기본 상위 **5건**.
- 더 있으면 마지막에 안내: `… 외 {n}건. 키워드를 좁혀보세요.`
```
🔎 "백엔드 · 서울 · 신입" 검색 결과 (총 12건 중 5건)

📌 ...
...
… 외 7건. 키워드를 좁혀보세요.
```

## 자동 푸시 (신규 공고 알림)
- 헤더로 "새 공고"임을 명시. 한 번에 너무 많으면 분할하거나 상위 N건만.
```
🆕 새 채용공고 {n}건

📌 ...
```

## 도움말 (`!채용도움`)
- 명령 문법과 예시를 간결하게.
```
사용법: !채용 [키워드] [지역:OO] [경력:신입|경력|무관] [형태:정규직|계약직]
예) !채용 백엔드 지역:서울 경력:신입
```

## 오류/빈 결과
- 결과 없음: `조건에 맞는 공고가 없어요. 키워드를 바꿔보세요.`
- 데이터 로드 실패: `공고 데이터를 불러오지 못했어요. 잠시 후 다시 시도해주세요.`
